Política de Privacidade - Alerta Bauzinho

Última atualização: 02 de julho de 2026

O aplicativo Alerta Bauzinho respeita a privacidade dos usuários e está comprometido com a proteção dos dados pessoais, em conformidade com a Lei Geral de Proteção de Dados (LGPD). Esta Política explica quais informações podem ser coletadas, como são utilizadas e quais são os direitos dos usuários.

1. Dados coletados

Durante a utilização do aplicativo poderão ser coletados:

  • Nome informado pelo usuário (quando aplicável);
  • Endereço de e-mail (quando informado pelo usuário);
  • Dados cadastrados no aplicativo;
  • Informações necessárias para o funcionamento da plataforma;
  • Dados técnicos de conexão necessários para o funcionamento dos serviços utilizados.

2. Finalidade da coleta

Os dados são utilizados exclusivamente para:

  • Permitir o funcionamento do aplicativo;
  • Armazenar informações cadastradas pelo usuário;
  • Gerenciar permissões de acesso;
  • Garantir a segurança da plataforma;
  • Corrigir problemas e melhorar continuamente o serviço.

3. Compartilhamento de informações

O Alerta Bauzinho não vende dados pessoais. As informações poderão ser processadas apenas pelos serviços essenciais utilizados pelo aplicativo, como os serviços fornecidos pela Google Firebase, necessários para o armazenamento dos dados.

4. Armazenamento dos dados

Os dados cadastrados pelo usuário são armazenados utilizando o Firebase Realtime Database, serviço fornecido pela Google, e permanecem armazenados enquanto forem necessários para o funcionamento do aplicativo ou até que o usuário solicite sua exclusão.

5. Segurança

São adotadas medidas técnicas e administrativas para proteger os dados contra acessos não autorizados, alterações, divulgação ou destruição indevida.

6. Exclusão de dados

O usuário poderá solicitar a exclusão de sua conta e dos dados armazenados enviando um e-mail para:

petrukio14@gmail.com

Após o recebimento da solicitação, os dados serão analisados e removidos dentro de prazo razoável, observadas eventuais obrigações legais.

7. Serviços de terceiros

O aplicativo utiliza os seguintes serviços:

  • Google Firebase Realtime Database.

8. Direitos do usuário

O usuário poderá solicitar:

  • Acesso aos seus dados;
  • Correção de informações;
  • Atualização dos dados;
  • Exclusão da conta;
  • Exclusão dos dados armazenados.

9. Crianças

O aplicativo não é destinado a menores de 13 anos e não coleta intencionalmente dados pessoais desse público.

10. Alterações desta Política

Esta Política poderá ser atualizada periodicamente. As alterações entrarão em vigor a partir de sua publicação nesta página.

11. Contato
